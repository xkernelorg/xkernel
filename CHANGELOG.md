# Changelog

## [0.1.0] — 2025-02-08
### Added
- Initial stable release of `xkernel`
- Deterministic X-field engine (`XKernel`, `XKernelConfig`)
- Waveform lens (`lens_wave.py`)
- Physics & information lenses (Schrödinger, Einstein, Shannon)
- ASCII bounce-dot viewer
- Full test suite
- MIT License
- Archived on Zenodo: https://doi.org/10.5281/zenodo.17859535

## [0.2.0] — 2025-12-08

- Added Einstein lens for curvature/strain/energy views of lattice states.
- Added Shannon and Schrödinger lenses for information and wave interpretations.
- Added analytic wave generator, ASCII wave viewer, and demo scripts.
- Expanded test suite and CI to cover lenses and visualization behavior.
- Cleaned build artifacts and aligned metadata for Zenodo archiving.
